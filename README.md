# Explore With Me (EWM)

Проект афиша. Здесь можно предложить какое-либо событие от выставки до похода в кино и собрать компанию для участия в нём.

## 📊 Статус разработки

### 🚦 Текущий этап: **1. Сервис статистики** (в активной разработке)

| Этап | Название                  | Статус       | Описание                                                   |
|------|---------------------------|--------------|------------------------------------------------------------|
| 1    | Сервис статистики         | 🔨 **В работе** | Реализация сервиса для сбора и анализа статистики посещений |
| 2    | Основной сервис           | ⏳ Впереди    | Главный функционал                               |
| 3    | Дополнительная функциональность | ⏳ Впереди | Фичи проекта                                               |

## Требования

- Java 21
- Docker и Docker Compose
- Maven 3.8+

## Запуск проекта

1. **Сборка проекта**:

```
mvn clean install
```
2. **Запуск контейнеров**:

```
docker-compose up --build
```
